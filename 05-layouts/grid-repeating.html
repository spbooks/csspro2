Repeating grids

In this example, we've created row and column track lists of 1fr, and indicated that these should repeat: grid: repeat(3, auto) / repeat(4, 1fr);. Notice that the auto in repeat(3, auto) means that the size of each row will expand or shrink to the length of the longest content, while repeat(4, 1fr) results in equally-proportioned columns.

Related specification: CSS Grid Layout Module Level 1.
